Understanding Collections in Programming

What are Collections?
Collections
Collections are containers that group multiple items into a single unit
Arrays
Arrays store elements in contiguous memory locations with fixed size
Lists
Lists are dynamic arrays that can grow or shrink in size
Sets
Sets store unique elements with no duplicates allowed
Maps (Dictionaries)
Maps store key-value pairs for quick lookups
Queues
Queues follow First-In-First-Out (FIFO) principle
Stacks
Stacks follow Last-In-First-Out (LIFO) principle
