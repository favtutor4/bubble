Sorting Algorithm Performance

Sorting Algorithm Time Complexity

This graph shows how different sorting algorithms perform as the input size increases. Notice that bubble sort, selection sort, and insertion sort grow much more quickly (they have O(n²) time complexity), while merge sort, quick sort, and heap sort grow more slowly (they have O(n log n) time complexity).

For small input sizes, all algorithms perform similarly, but as the input grows, the differences become dramatic!
