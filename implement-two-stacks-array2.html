Dynamic Two Stacks Implementation

Introduction to Dynamic Two Stacks
Dynamic Two Stacks implement two stacks in a single array efficiently. Instead of dividing the array in half, both stacks can grow dynamically toward each other.

We start with an empty array where both stacks can grow. Stack 1 grows from left to right, and Stack 2 grows from right to left.
Initializing the Stacks
Initially, top1 starts at -1 (left side) and top2 starts at array length (right side). Both stacks are empty at the beginning.

Stack Overflow Condition
A stack overflow occurs only when both stacks meet in the middle (when top1 + 1 = top2).

The dynamic approach is efficient for:
- Memory utilization (no wasted space)
- Flexibility (stacks can grow as needed)
- Space optimization (no fixed division)
- Array reuse (single array for two stacks)
